# 🏥 MedisynX - Next Gen AI Healthcare Portal

![Banner](https://img.shields.io/badge/MedisynX-AI--Powered-00f2fe?style=for-the-badge)
![Gemini 2.5](https://img.shields.io/badge/Powered%20By-Gemini%202.5%20Flash-blueviolet?style=for-the-badge)

MedisynX is a cutting-edge, full-stack MERN healthcare management platform designed for modern clinical environments. It leverages the raw power of **Gemini 2.5 Flash** to provide clinicians with high-throughput, AI-assisted diagnostic insights.

## ✨ Premium Features

- 🧠 **AI Clinical Decision Support**: Real-time analysis of symptoms using Gemini 2.5 Flash.
- 💎 **Glassmorphism UI**: A stunning, high-performance dashboard built with modern design principles.
- 👥 **Advanced Patient Management**: Secure and intuitive patient tracking and history.
- 📅 **Smart Scheduling**: Seamless appointment management for busy practices.
- 💳 **Integrated Billing**: Professional invoice generation and payment tracking.
- 🚀 **Guest Demo Mode**: Instant access to experience the platform's capabilities.

## 🛠️ Tech Stack

- **Frontend**: React 19, Vite, Lucide Icons, React Markdown.
- **Backend**: Node.js, Express 5.
- **AI**: Google Generative AI (Gemini 2.5 Flash).
- **Database**: MongoDB Atlas.
- **Styling**: Vanilla CSS with a custom Glassmorphism design system.

## 🚀 Getting Started

### Prerequisites
- Node.js installed.
- MongoDB Atlas account.
- Google AI Studio API Key.

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/ManakRaj-7/MedisynX.git
   cd MedisynX
   ```

2. **Setup Server**
   ```bash
   cd server
   npm install
   # Create .env and add MONGODB_URI and GEMINI_API_KEY
   npm run dev
   ```

3. **Setup Client**
   ```bash
   cd ../client
   npm install
   npm run dev
   ```
